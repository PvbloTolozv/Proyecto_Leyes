## ¿Es posible predecir si una ley tiene chances de ser aprobada o no?
<li> Para esta incógnita he generado un modelo de Machine Learning que permite predecirlo con hasta un 81% de precision. </li>
<li> Con una base de datos que contiene información de los proyectos de ley en un periodo de 20 años logramos en base a un gran preprocesamiento de la BDD identificar las variables relevantes, agregamos un par de variables en base a la literatura científica y procesamos los titulos de cada ley para que nos entregaran información util en el desarrollo. </li>
<li> Este proyecto fue parte de nuestro Trabajo final en la academia Desafíos Latam. </li>
